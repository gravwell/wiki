# Changelog for version 5.9.2

## Released 7 July 2026

## Gravwell

### Additions

* Added ARM64 builds.
* Added link to AI usage limits documentation. 

### Bug Fixes

* Fixed an issue where the Logbot UI would stall on empty completion requests.
* Fixed an issue where the runtime of flow nodes and scripts was double logged.
* Fixed an issue with uploading files using the Slack File flow node.
* Improved Logbot flow node timeouts.

## Ingester Changes

### Additions

* Added ARM64 builds.
* Added targeted debugging for the HTTP ingester.
* Added hosted ingester logs to the `hosted_ingester.log` file.

### Bug Fixes

* Fixed an issue where the MacOS File Follower was leaking file descriptors on closed files.
* Fixed an issue where the Hosted Runner had the ingest-secret commented out on install.
* Fixed an issue where the Hosted Runner could not validate the config while the systemd service was up. 
* Reduced the group membership for the Hosted Runner.
