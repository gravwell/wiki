# Changelog for version 5.9.1

## Released 17 June 2026

## Gravwell

### Additions

* Added [`Max-AI-Tokens`](max-ai-tokens) as a configurable option for AI.
* Added centroid coordinates as an optional output for the kmeans module.

### Bug Fixes

* Fixed an issue where the Logbot chat spinner remained active after hitting the max token limit.
* Fixed an issue where the Logbot flow node would not execute MCP tool calls.
* Fixed an issue with a decode error in the Logbot flow node.
* Fixed an issue with a missing tooltip when Logbot was disabled by CBAC.
* Fixed an issue with the kmeans module not emitting block stats.
* Fixed an issue with the permissive mode drop logic in the regex module for entries with a non-existent or empty EV.
* Fixed an issue with broken filtering on instrinsic values with no given module.
* Improved error handling for the LLM proxy.

## Ingester Changes

### Bug Fixes

* Fixed an issue where the Azure Event Hubs ingester would crash when a bad timegrinder spec was configured.
* Set a better default for the gravwellGenerator max entry size.
