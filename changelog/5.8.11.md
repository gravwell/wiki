# Changelog for version 5.8.11

## Released 29 January 2026

## Gravwell

```{attention}
This release contains several high priority bug fixes to address security concerns.
```

### Bug Fixes

* Disabled additional network functions in anko.
* Disabled scripts on user-provided file types.
* Fixed an issue where cached scheduled searches were not updated when write access changes were made. 
* Implemented whitelist for file upload content types.
* Improved session cookie security. 
* Removed security sensitive packages (machineid, freeport, viper, goproxy, httpgzip) from the Go Flow node.
* Updated searchagent to better secure automation container when misconfigured to run as root.
